Smart Chef 🍳

Smart Chef ایک AI-powered recipe app ہے جو اردو اور انگلش میں cooking کو آسان بناتا ہے۔

## Features
- رجسٹریشن اور لاگ ان سسٹم
- Voice Search (🎤)
- Live Cooking Mode (⏱️)
- Offline Support (PWA)
- Urdu Accessibility

## How to Run
1. Clone repo
2. Open index.html in browser
3. Register & Login to start cooking!
